## 📦 Developer info

[![Automatic tests level](https://apps.yunohost.org/badge/cilevel/docker_gate)](https://ci-apps.yunohost.org/ci/apps/docker_gate/)

🛠️ Upstream Docker Gate repository: <https://github.com/byrtn/docker_gate_ynh>

Pull request are welcome and should target the [`testing` branch](https://github.com/byrtn/docker_gate_ynh/tree/testing).

The `testing` branch can be tested using:
```
# fresh install:
sudo yunohost app install https://github.com/byrtn/docker_gate_ynh/tree/testing

# upgrade an existing install:
sudo yunohost app upgrade docker_gate -u https://github.com/byrtn/docker_gate_ynh/tree/testing
```

You can also switch to the testing branch to update from testing by default (as same as for APT when you chose to use a testing repos) with this command:
```bash
sudo yunohost app setting docker_gate upgrade_channel -v testing
```
